# 🛒 E-Commerce Shopping Behavior Analysis

## 📌 Project Overview

This project analyzes **3,900 customer purchase records** to understand shopping behavior, customer demographics, purchasing patterns, product preferences, and revenue trends. The analysis combines **Python, SQL, and Power BI** to transform raw data into actionable business insights through data cleaning, exploratory data analysis (EDA), feature engineering, SQL analysis, and interactive dashboards.

The objective is to help businesses improve customer retention, optimize product offerings, and make data-driven marketing and sales decisions.

---

## 🎯 Objectives

- Analyze customer purchasing behavior.
- Identify high-value customer segments.
- Discover top-selling and top-rated products.
- Evaluate the impact of subscriptions, discounts, and shipping methods.
- Build an interactive dashboard for business reporting.
- Generate actionable business recommendations.

## 📊 Project Workflow

### Data Cleaning
- Inspected dataset structure and data types.
- Renamed columns using `snake_case`.
- Removed redundant columns.
- Handled missing values.
- Validated discount-related fields.

### Feature Engineering
- Created Age Group and Purchase Frequency features.
- Imputed missing review ratings using category median.

### Exploratory Data Analysis
- Customer demographics
- Revenue analysis
- Product popularity
- Review ratings
- Subscription behavior
- Shipping methods

- Seasonal trends

### SQL Analysis

## 🔍 Key Insights

- Loyal customers form the largest customer segment.
- Clothing generates the highest overall revenue.
- Express shipping customers spend slightly more on average.
- Gloves, Sandals, Boots, Hats, and Handbags receive the highest ratings.
- Some products are highly discount-sensitive.
- Category-wise best sellers can improve merchandising strategies.
